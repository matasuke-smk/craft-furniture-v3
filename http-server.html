🏠 クラフトファニチャー ローカルサーバー

✅ 準備完了！以下の手順でサーバーを起動してください

📝 手順1: コマンドプロンプトを開く

以下のいずれかの方法でコマンドプロンプトを開いてください：

  • 方法A: Windows キー + R → cmd と入力 → Enter
  • 方法B: スタートメニューで「cmd」と検索 → 「コマンドプロンプト」をクリック
  • 方法C: Windows キー + X → 「ターミナル」または「コマンドプロンプト」を選択

📁 手順2: ディレクトリに移動

以下のコマンドをコピー&ペーストして実行：

cd /d "C:\Users\shioz\iCloudDrive\portfolio\craft-furniture"

🚀 手順3: サーバーを起動

以下のコマンドをコピー&ペーストして実行：

node server-alt.js

※ ポート8080でサーバーが起動します

🌐 手順4: ブラウザでアクセス

サーバーが起動したら、新しいタブで以下のURLにアクセス：

http://localhost:8080

⚠️ うまくいかない場合

エラーが出る場合は、以下を試してください：

  • 他のポート番号を試す: node simple-server.js (ポート3000)
  • Node.jsがインストールされているか確認: node --version
  • ファイルを直接開く: エクスプローラーで index.html をブラウザにドラッグ

📋 利用可能なページ一覧

サーバー起動後、以下のページにアクセスできます：

  • トップページ - メインページ
  • 施工事例 - フィルター機能付き
  • 職人紹介 - 3名のプロフィール
  • 見積もりシミュレーター - 価格計算
  • ショールーム - 予約システム
  • お知らせ - ニュース一覧
  • 納品エリア - 配送情報
  • プライバシーポリシー - 法的文書

🎉 完了！

サーバーが起動したら、クラフトファニチャーのウェブサイトの全機能をローカル環境でお試しいただけます。

停止するには: コマンドプロンプトで Ctrl+C を押してください。
